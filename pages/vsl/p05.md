We will use the classic **Iris dataset** for illustration.

### **Step 1 â€” Setup: Load Libraries and Dataset**

```python
import matplotlib.pyplot as plt
import seaborn as sns
import pandas as pd

# Load Iris dataset from seaborn
iris = sns.load_dataset('iris')

# Take a quick look
print(iris.head())
```

---

### **Step 2 â€” Basic Scatter Plot with Matplotlib**

We will first make a simple scatter plot using only `matplotlib`.

```python
plt.figure(figsize=(6, 4))

plt.scatter(iris['sepal_length'], iris['sepal_width'], color='blue')

plt.title('Basic Scatter Plot')
plt.xlabel('Sepal Length (cm)')
plt.ylabel('Sepal Width (cm)')

plt.show()
```

âœ… **What we have done:**

* Created a basic scatter plot
* Added title and labels

---

### **Step 3 â€” Refine with Categories (Species)**

Letâ€™s add color-coding based on species.

```python
plt.figure(figsize=(6, 4))

species = iris['species'].unique()
colors = ['red', 'green', 'blue']

for sp, color in zip(species, colors):
    subset = iris[iris['species'] == sp]
    plt.scatter(subset['sepal_length'], subset['sepal_width'], label=sp, color=color)

plt.title('Scatter Plot with Categories')
plt.xlabel('Sepal Length (cm)')
plt.ylabel('Sepal Width (cm)')
plt.legend()
plt.show()
```

âœ… **What we improved:**

* Color-coded each species
* Added legend

---

### **Step 4 â€” Switch to Seaborn for Cleaner Code**

Seaborn makes it much easier:

```python
plt.figure(figsize=(6, 4))
sns.scatterplot(data=iris, x='sepal_length', y='sepal_width', hue='species', palette='Set1')
plt.title('Scatter Plot with Seaborn')
plt.show()
```

âœ… **Seaborn advantage:**

* Automatically handles colors and legend

---

### **Step 5 â€” Add Styling and Improve Aesthetics**

Add better markers, larger points, transparency (`alpha`), and grid.

```python
plt.figure(figsize=(8, 6))
sns.scatterplot(
    data=iris, 
    x='sepal_length', 
    y='sepal_width', 
    hue='species', 
    palette='Set1',
    s=100,       # size of points
    style='species',  # different marker styles per species
    alpha=0.7    # transparency
)
plt.title('Refined Scatter Plot')
plt.grid(True)
plt.show()
```

---

### **Step 6 â€” Add Regression Line (Optional)**

If you want to add a linear regression line:

```python
plt.figure(figsize=(8, 6))
sns.lmplot(
    data=iris, 
    x='sepal_length', 
    y='sepal_width', 
    hue='species', 
    palette='Set1',
    height=6,
    aspect=1.2
)
plt.title('Scatter Plot with Regression Line')
plt.show()
```

âœ… **Note:** `lmplot()` adds regression lines automatically.

---

### **Step 7 â€” Final: Fully Polished Scatter Plot**

We combine everything into a fully polished scatter plot.

```python
plt.figure(figsize=(10, 7))
sns.scatterplot(
    data=iris,
    x='sepal_length',
    y='sepal_width',
    hue='species',
    style='species',
    s=150,
    palette='Set2',
    alpha=0.8,
    edgecolor='black'
)

plt.title('Final Polished Scatter Plot', fontsize=16)
plt.xlabel('Sepal Length (cm)', fontsize=12)
plt.ylabel('Sepal Width (cm)', fontsize=12)
plt.grid(True)
plt.legend(title='Species')
plt.show()
```
