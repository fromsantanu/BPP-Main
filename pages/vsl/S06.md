### üß∞ **Setup: Install and Import Libraries**

```python
import pandas as pd
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt
```

* **pandas (pd):** for data manipulation.
* **numpy (np):** for generating sample data.
* **seaborn (sns):** for heatmaps.
* **matplotlib.pyplot (plt):** for additional customization.

---

## Step 1 ‚Äî Create Sample Data

We‚Äôll create a simple correlation matrix to visualize.

```python
# Generate sample data
np.random.seed(42)

data = pd.DataFrame({
    'Sales': np.random.normal(200, 30, 100),
    'Advertising': np.random.normal(50, 10, 100),
    'Customer_Satisfaction': np.random.normal(80, 5, 100),
    'Profit': np.random.normal(20, 5, 100)
})

# Calculate correlation matrix
corr_matrix = data.corr()

print(corr_matrix)
```

#### Explanation:

* We have 4 continuous variables.
* `data.corr()` calculates Pearson correlation coefficients between columns.

---

## Step 2 ‚Äî Basic Heatmap

```python
# Basic heatmap
sns.heatmap(corr_matrix)
plt.show()
```

### üîç Code Breakdown:

* `sns.heatmap()` ‚Üí Seaborn function for heatmaps.
* `corr_matrix` ‚Üí The matrix data to visualize.

---

## Step 3 ‚Äî Improve Figure Size

```python
plt.figure(figsize=(8, 6))
sns.heatmap(corr_matrix)
plt.show()
```

* `plt.figure(figsize=(8, 6))` ‚Üí Increases the size of the plot.

---

## Step 4 ‚Äî Add Color Bar and Annotations

```python
plt.figure(figsize=(8, 6))
sns.heatmap(corr_matrix, annot=True, cmap='coolwarm')
plt.title('Correlation Matrix Heatmap', fontsize=16)
plt.show()
```

* `annot=True` ‚Üí Writes numeric values inside each cell.
* `cmap='coolwarm'` ‚Üí Changes color palette to visually differentiate positive/negative correlations.
* `plt.title()` ‚Üí Adds title.

---

## Step 5 ‚Äî Control Number Formatting

```python
plt.figure(figsize=(8, 6))
sns.heatmap(corr_matrix, annot=True, fmt='.2f', cmap='coolwarm')
plt.title('Correlation Matrix (2 Decimal Places)', fontsize=16)
plt.show()
```

* `fmt='.2f'` ‚Üí Formats numbers to 2 decimal places.

---

## Step 6 ‚Äî Customize Color Scale (Centering and Range)

```python
plt.figure(figsize=(8, 6))
sns.heatmap(corr_matrix, annot=True, fmt='.2f', cmap='coolwarm', center=0, vmin=-1, vmax=1)
plt.title('Centered Correlation Heatmap', fontsize=16)
plt.show()
```

* `center=0` ‚Üí Centers color around zero.
* `vmin` and `vmax` ‚Üí Fix value range from -1 to 1 (correlation coefficient range).

---

## Step 7 ‚Äî Add Line Separators (Gridlines)

```python
plt.figure(figsize=(8, 6))
sns.heatmap(corr_matrix, annot=True, fmt='.2f', cmap='coolwarm', linewidths=0.5, linecolor='black')
plt.title('Heatmap with Gridlines', fontsize=16)
plt.show()
```

* `linewidths=0.5` ‚Üí Thickness of gridlines.
* `linecolor='black'` ‚Üí Color of gridlines.

---

## Step 8 ‚Äî Remove Color Bar (Optional)

```python
plt.figure(figsize=(8, 6))
sns.heatmap(corr_matrix, annot=True, fmt='.2f', cmap='coolwarm', cbar=False)
plt.title('Heatmap without Color Bar', fontsize=16)
plt.show()
```

* `cbar=False` ‚Üí Hides the color bar.

---

## Step 9 ‚Äî Rotate Axis Labels for Better Readability

```python
plt.figure(figsize=(8, 6))
sns.heatmap(corr_matrix, annot=True, fmt='.2f', cmap='coolwarm')
plt.xticks(rotation=45)
plt.yticks(rotation=0)
plt.title('Rotated Axis Labels', fontsize=16)
plt.show()
```

* `plt.xticks(rotation=45)` ‚Üí Rotates X-axis labels 45 degrees.
* `plt.yticks(rotation=0)` ‚Üí Keeps Y-axis labels horizontal.

---

## Step 10 ‚Äî Use Mask to Hide Upper (or Lower) Triangle (For Symmetric Matrices)

```python
# Create mask
mask = np.triu(np.ones_like(corr_matrix, dtype=bool))

plt.figure(figsize=(8, 6))
sns.heatmap(corr_matrix, annot=True, fmt='.2f', cmap='coolwarm', mask=mask)
plt.title('Heatmap (Lower Triangle Only)', fontsize=16)
plt.show()
```

* `mask=mask` ‚Üí Hides upper triangle of the symmetric matrix.
* `np.triu()` ‚Üí Creates a mask for upper triangle.
