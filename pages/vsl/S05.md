### üß∞ **Setup: Install and Import Libraries**

```python
import pandas as pd
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt
```

* **pandas (pd):** for data manipulation.
* **numpy (np):** for generating random data.
* **seaborn (sns):** for creating scatter plots.
* **matplotlib.pyplot (plt):** for plot control and customization.

---

## Step 1 ‚Äî Create Sample Data

We‚Äôll simulate some simple data.

```python
# Generate sample data
np.random.seed(42)

data = pd.DataFrame({
    'Advertising': np.random.uniform(100, 500, 200),  # Budget spent on ads
    'Sales': np.random.normal(loc=300, scale=50, size=200)
})

print(data.head())
```

#### Explanation:

* `Advertising` ‚Üí Independent variable (X-axis).
* `Sales` ‚Üí Dependent variable (Y-axis).
* This data will help us visualize potential correlation.

---

## Step 2 ‚Äî Basic Scatter Plot

```python
# Basic scatter plot
sns.scatterplot(data=data, x='Advertising', y='Sales')
plt.show()
```

### üîç Code Breakdown:

* `sns.scatterplot()` ‚Üí Seaborn function for scatter plots.
* `data=data` ‚Üí The dataset.
* `x='Advertising'` ‚Üí X-axis.
* `y='Sales'` ‚Üí Y-axis.

---

## Step 3 ‚Äî Improve Figure Size

```python
plt.figure(figsize=(8, 5))
sns.scatterplot(data=data, x='Advertising', y='Sales')
plt.show()
```

* `plt.figure(figsize=(8, 5))` ‚Üí Bigger figure size.

---

## Step 4 ‚Äî Add Title and Axis Labels

```python
plt.figure(figsize=(8, 5))
sns.scatterplot(data=data, x='Advertising', y='Sales')
plt.title('Sales vs Advertising Spend', fontsize=16)
plt.xlabel('Advertising Budget ($)', fontsize=12)
plt.ylabel('Sales (Units)', fontsize=12)
plt.show()
```

* Titles and labels help clarify what the chart represents.

---

## Step 5 ‚Äî Add Color to Points

```python
plt.figure(figsize=(8, 5))
sns.scatterplot(data=data, x='Advertising', y='Sales', color='darkorange')
plt.title('Sales vs Advertising Spend', fontsize=16)
plt.show()
```

* `color='darkorange'` ‚Üí Sets all points to same color.

---

## Step 6 ‚Äî Add Category for Hue (Multiple Groups)

Let‚Äôs introduce a categorical variable:

```python
# Add Region
data['Region'] = np.random.choice(['North', 'South', 'East', 'West'], size=200)

plt.figure(figsize=(8, 5))
sns.scatterplot(data=data, x='Advertising', y='Sales', hue='Region')
plt.title('Sales vs Advertising Spend by Region', fontsize=16)
plt.show()
```

* `hue='Region'` ‚Üí Automatically color-codes points by category.
* Seaborn automatically adds a legend.

---

## Step 7 ‚Äî Change Markers and Size

```python
plt.figure(figsize=(8, 5))
sns.scatterplot(data=data, x='Advertising', y='Sales', hue='Region',
                style='Region', s=100)
plt.title('Sales vs Advertising Spend by Region', fontsize=16)
plt.show()
```

* `style='Region'` ‚Üí Assigns different markers for each category.
* `s=100` ‚Üí Controls the size of the markers.

---

## Step 8 ‚Äî Add Additional Numeric Variable with Size Encoding

Let‚Äôs add one more numeric feature:

```python
# Add another numeric variable: Customer Visits
data['Visits'] = np.random.randint(10, 100, size=200)

plt.figure(figsize=(8, 5))
sns.scatterplot(data=data, x='Advertising', y='Sales',
                hue='Region', size='Visits', sizes=(20, 200), alpha=0.7)
plt.title('Sales vs Advertising Spend (Size = Visits)', fontsize=16)
plt.show()
```

* `size='Visits'` ‚Üí Marker size represents number of customer visits.
* `sizes=(20, 200)` ‚Üí Range of marker sizes.
* `alpha=0.7` ‚Üí Transparency level (avoid clutter).

---

## Step 9 ‚Äî Add Regression Line (Trendline)

```python
plt.figure(figsize=(8, 5))
sns.regplot(data=data, x='Advertising', y='Sales', scatter_kws={'color':'skyblue'}, line_kws={'color':'red'})
plt.title('Sales vs Advertising Spend with Regression Line', fontsize=16)
plt.show()
```

* `sns.regplot()` ‚Üí Same as scatterplot but adds linear regression line.
* `scatter_kws` ‚Üí Styling for scatter points.
* `line_kws` ‚Üí Styling for regression line.

---

## Step 10 ‚Äî Customize Seaborn Theme

```python
sns.set_style("whitegrid")

plt.figure(figsize=(8, 5))
sns.scatterplot(data=data, x='Advertising', y='Sales',
                hue='Region', size='Visits', sizes=(20, 200), alpha=0.7)
plt.title('Styled Scatter Plot', fontsize=16)
plt.show()
```

* `sns.set_style("whitegrid")` ‚Üí Changes background appearance.
* Options: `'whitegrid'`, `'darkgrid'`, `'white'`, `'dark'`, `'ticks'`.
